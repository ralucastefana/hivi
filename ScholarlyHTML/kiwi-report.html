Introducere

KIWI este o extensie de browser Web capabilă să ofere utilizatorului maniere multiple de vizualizare a istoricului navigării şi a adreselor Web favorite (bookmark-uri) locale sau preluate din surse externe precum Pocket. Reprezentările grafice generate vor putea fi exportate în formatele PNG şi SVG. Instrumentul va putea fi configurat – de exemplu, excluderea unor domenii Internet sau URL-uri particulare de la vizualizare, stabilirea atributelor zonei în care se va produce vizualizarea, oferirea de teme cromatice etc.

Scop

Scopul acestei extensii este de a oferi o statistică asupra istoricului navigării unei persoane, oferind într-o formă atractivă vizual aceste date, pentru a putea fi mai ușor interpretate și prelucrate.

Concluzie

Web Browser History Visualizer este o aplicație axată pe controlul asupra site-urilor web accesate. Acest tip de plugin este util atunci cand se doreste obtinerea unui track al fluxului de accesari si poate fi un bun indicator al managementului timpului.
